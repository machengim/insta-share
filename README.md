# isend

Send files over LAN. Inspired by [Send-anywhere](https://send-anywhere.com/#transfer).

Works on Linux and MacOS.

### Quick start:

    isend -s -m "Hello, I'm from isend" a.txt b.png
    isend -r your_code

### Help:

    isend --help

Implemented features:
+ Send: -s
+ Receive: -r
+ Message: -m
+ Password: -p
